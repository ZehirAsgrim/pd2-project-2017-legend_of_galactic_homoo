# わが征くはホモォの大海
![](https://i.imgur.com/lalkMSG.jpg)
以C++撰寫的Qt遊戲，玩法類似《部落衝突：皇室戰爭》。各種梗衍伸自銀河英雄傳說。
### Feature
#### Magic
* 雷神之錘
伊謝爾倫要塞每隔一段長時間可發射 消滅路徑上所有角色![](https://i.imgur.com/24qT5ol.png)
* 機雷
角色碰觸會爆炸 消滅一定半徑範圍內的敵人
#### Manage cards page
![](https://i.imgur.com/UJU8j89.jpg)
可選擇英雄以及其他友方角色
#### Hero system 
可自由操控英雄
